Public one-pager for **LEO**, the agentic-first content factory for multiple brands by **498A × ZOOPA**.

🌐 **Live**: https://498as.github.io/leo-onepager/

## About

LEO is the first content generator that combines:
- **Agentic-first generation** (JSON-LD, citable claims, entity index by default)
- **Process-driven workflow** (6 acts that mirror the editor's mental flow)
- **Multi-brand isolation** as a foundational architecture principle
- **Learning loop per brand** (VoiceExemplars)
- **Proactive topic discovery** (daily agent across 8 sources)

Status: PRD v2.0 Greenfield · Spec ready for Figma wireframe phase · Timeline 20–26 weeks · UX-first.
